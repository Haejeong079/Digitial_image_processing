Week 5 · 어두운 이미지를 선명하게 만드는 방법

👣 단계별로 이해하기
🧮 공식 파헤치기
h(v) = round( (CDF(v) - CDF_min) / (CDF_max - CDF_min) × 255 )
변수가 헷갈리죠? 아래 표에서 하나씩 확인해보세요.
변수뜻예시 (8픽셀 기준)
v지금 처리 중인 픽셀 값 (0~255)v = 50 → 밝기 50인 픽셀
CDF(v)v 이하인 픽셀의 누적 합계CDF(50) = 3 → 50 이하 픽셀 3개
CDF_minCDF 중 0이 아닌 가장 작은 값첫 번째로 등장하는 픽셀의 CDF 값
CDF_max전체 픽셀 수픽셀이 8개면 CDF_max = 8
255최대 밝기 (L-1)흑백 이미지 최댓값
비유: 공식은 "현재 픽셀이 전체에서 몇 번째로 밝은가?"를 0~255 사이로 다시 표현하는 것이에요!
✏️ 직접 계산해보는 예시 — 8개 픽셀
픽셀 값: 50, 50, 80, 80, 80, 120, 200, 200 (어두운 쪽에 몰려있는 상황)
픽셀값 v개수CDF(v)계산 과정결과 h(v)
평활화 후 픽셀:
같은 비율 관계를 유지하면서 전체 0~255 범위로 쭉 펼쳐졌어요!
💻 코드로 구현하기 (과제 핵심!)
gray = cv2.imread('bean.jpg', cv2.IMREAD_GRAYSCALE) # Step 1: 히스토그램 계산 hist, _ = np.histogram(gray.flatten(), 256, [0, 256]) # Step 2: CDF 계산 (누적합) cdf = hist.cumsum() # Step 3: CDF_min (0 제외한 최솟값) cdf_min = cdf[cdf > 0].min() cdf_max = cdf.max() # = 전체 픽셀 수 # Step 4: 공식 적용 → LUT(변환표) 만들기 lut = np.round( (cdf - cdf_min) / (cdf_max - cdf_min) * 255 ).astype(np.uint8) # Step 5: 원본 이미지에 LUT 적용 result = lut[gray]
cv2.equalizeHist() 쓰면 미제출 처리! 반드시 위 공식으로 직접 구현해야 해요.
