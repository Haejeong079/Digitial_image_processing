Nearest Neighbor vs Bilinear 비교
Nearest Neighbor — 계단 현상
Bilinear — 부드럽게 섞임
Nearest Neighbor는 가장 가까운 픽셀을 그냥 복사 → 계단 현상 발생. Bilinear는 4개 픽셀을 거리에 따라 섞음 → 훨씬 부드러운 결과!
OpenCV 코드
result = cv2.resize( img, (w * 2, h * 2), interpolation=cv2.INTER_LINEAR # Bilinear )
cv2.INTER_NEAREST = Nearest Neighbor (빠르지만 계단) · cv2.INTER_LINEAR = Bilinear (부드러움) · cv2.INTER_CUBIC = Bicubic (더 부드럽지만 느림)
