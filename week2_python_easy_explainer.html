반복문 — for / while 헷갈리지 말자!
for 문 "이만큼 반복해!"
비유: "1~9까지 번호표 뽑아서 한 명씩 차례대로 처리해!" 같은 것
for n in range(1, 10): print(n) # 1, 2, 3, ... 9 출력 for n in range(0, 10, 2): print(n) # 0, 2, 4, 6, 8 출력 fruits = ['사과', '바나나', '딸기'] for f in fruits: print(f)
range(1, 10) 은 1~9까지! 10은 포함 안 됨 (슬라이싱과 동일한 규칙)
while 문 "조건이 참인 동안 반복해!"
비유: "신호등이 빨간불인 동안 계속 기다려!" 같은 것
a = 1 while a < 10: print(a) a += 1 # 이걸 빠뜨리면 무한루프!
while 문 안에서 조건을 바꾸는 코드를 빠뜨리면 무한 루프에 빠집니다!
함수(def) "재사용 가능한 코드 묶음"
def greet(name): msg = "안녕, " + name return msg result = greet("민수") print(result) # → 안녕, 민수 def power(x, n=2): # n의 기본값은 2 return x ** n print(power(3)) # → 9 print(power(3, 3)) # → 27
비유: 함수 = 레시피. 재료(인자)를 넣으면 음식(결과)이 나오는 것!
함수 안에서 만든 변수는 함수 밖에서 사용 불가 (함수가 끝나면 사라짐)
