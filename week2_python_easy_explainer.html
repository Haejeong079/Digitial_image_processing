Matplotlib으로 보여줄 땐 반드시 BGR→RGB 변환 필요!
import cv2, matplotlib.pyplot as plt img = cv2.imread('image.jpg') # BGR로 읽힘 # 방법 1: cvtColor로 변환 rgb = cv2.cvtColor(img, cv2.COLOR_BGR2RGB) plt.imshow(rgb) # 정상 색상 # 방법 2: 슬라이싱으로 변환 rgb = img[:, :, ::-1] plt.imshow(rgb)
비유: OpenCV는 "파빨초" 순서로 읽는 특이한 카메라. 보여줄 때는 "빨초파"로 다시 뒤집어야 해요!

반복문 — for / while 헷갈리지 말자!
for 문 "이만큼 반복해!"
비유: "1~9까지 번호표 뽑아서 한 명씩 차례대로 처리해!" 같은 것
for n in range(1, 10): print(n) # 1, 2, 3, ... 9 출력 # range(시작, 끝, 간격) for n in range(0, 10, 2): print(n) # 0, 2, 4, 6, 8 출력 # 리스트 순회 fruits = ['사과', '바나나', '딸기'] for f in fruits: print(f) # 사과, 바나나, 딸기 차례로 출력
range(1, 10) 은 1~9까지! 10은 포함 안 됨 (슬라이싱과 동일한 규칙)
while 문 "조건이 참인 동안 반복해!"
비유: "신호등이 빨간불인 동안 계속 기다려!" 같은 것
a = 1 while a < 10: print(a) a += 1 # a = a + 1 (이걸 빠뜨리면 무한루프!) # for문과 같은 결과: 1, 2, 3, ... 9 출력
while 문 안에서 조건을 바꾸는 코드를 빠뜨리면 무한 루프에 빠집니다!
함수(def) "재사용 가능한 코드 묶음"
def greet(name): # 함수 정의 msg = "안녕, " + name return msg # 결과 반환 result = greet("민수") # 함수 호출 print(result) # → 안녕, 민수 # 기본값(default) 설정 def power(x, n=2): # n의 기본값은 2 return x ** n print(power(3)) # → 9 (n=2로 자동) print(power(3, 3)) # → 27 (n=3으로 지정)
비유: 함수 = 레시피. 재료(인자)를 넣으면 음식(결과)이 나오는 것. 한 번 만들어두면 계속 재사용 가능!
함수 안에서 만든 변수는 함수 밖에서 사용 불가 (함수가 끝나면 사라짐)
