이미지 = 조명 × 반사율. 곱셈이라서 따로 분리가 불가능해요. 그런데 log를 씌우면?

log 이후 (덧셈 — 분리 가능!)
ln(f) = ln(i) + ln(r)
주파수 도메인에서
따로 처리 가능!
f(x,y) = i(x,y) · r(x,y) ln(f) = ln(i) + ln(r) ← 이제 따로 처리 가능! H(u,v) = (γH - γL)(1 - e^(-cD²/D₀²)) + γL γL < 1 (조명 약하게), γH > 1 (반사율 강하게)

💻 코드 구현
img = cv2.imread('sample.jpg', cv2.IMREAD_GRAYSCALE).astype(float) # Step 1: log 변환 (0 방지 위해 1 더함) z = np.log(img + 1) # Step 2: DFT + centering M, N = z.shape fp = np.zeros((2*M, 2*N)) fp[:M, :N] = z * ((-1)**np.indices((M,N)).sum(axis=0)) Z = np.fft.fft2(fp) # Step 3: Homomorphic filter u = np.arange(2*M) - M v = np.arange(2*N) - N V, U = np.meshgrid(v, u) D2 = U**2 + V**2 gL, gH, c, D0 = 0.5, 2.0, 1, 30 H = (gH - gL) * (1 - np.exp(-c * D2 / D0**2)) + gL # Step 4: 필터 적용 후 IDFT S = np.real(np.fft.ifft2(H * Z))[:M, :N] S = S * ((-1)**np.indices((M,N)).sum(axis=0)) # Step 5: exp 복원 result = np.exp(S) - 1 result = np.clip(result, 0, 255).astype(np.uint8)
