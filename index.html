Week 4 · Sampling & Upsampling (보간법)
핵심 요약
Nyquist 정리, Zone Plate, 보간법(Nearest Neighbor / Bilinear / Bicubic)을 다룹니다.
🌊 저주파 vs 고주파
🌊 저주파 = 천천히 변하는 부분 (배경, 하늘) / 고주파 = 빠르게 변하는 부분 (엣지, 경계선)
⭐ Nyquist 정리
Stride(간격)가 Nyquist frequency보다 크면 → 앨리어싱(aliasing) 발생! 정보 왜곡!
Zone Plate 이미지로 확인: stride를 크게 하면 바깥쪽(고주파) 영역이 뭉개짐
⭐ 보간법 (Interpolation) 핵심
📸 작은 이미지를 크게 키울 때 없는 픽셀을 채우는 방법!
Nearest Neighbor
가장 가까운 픽셀 값 그대로 복사. 빠르지만 계단 현상
Bilinear
주변 4개 점을 선형 보간. 더 부드러운 결과
Bicubic
주변 16개 점으로 3차 함수 보간. 가장 부드러움
OpenCV 코드
cv2.resize(img, (w,h),
interpolation=cv2.INTER_LINEAR)

import cv2

실습 4 · 수동 Bilinear 보간 vs cv2.resize
import numpy as np
import cv2

img = cv2.imread('pandas.jpg', cv2.IMREAD_GRAYSCALE).astype(float)
H, W   = img.shape
newH, newW = H * 2, W * 2

# ── 수동 Bilinear ──────────────────────────────────
result_manual = np.zeros((newH, newW))
for ny in range(newH):
    for nx in range(newW):
        ox = nx * (W - 1) / (newW - 1)
        oy = ny * (H - 1) / (newH - 1)
        x0, y0 = int(ox), int(oy)
        x1 = min(x0 + 1, W - 1)
        y1 = min(y0 + 1, H - 1)
        tx, ty = ox - x0, oy - y0
        r1 = img[y0, x0] * (1 - tx) + img[y0, x1] * tx
        r2 = img[y1, x0] * (1 - tx) + img[y1, x1] * tx
        result_manual[ny, nx] = r1 * (1 - ty) + r2 * ty
result_manual = result_manual.astype(np.uint8)

# ── OpenCV Bilinear ────────────────────────────────
result_cv2 = cv2.resize(
    img.astype(np.uint8), (newW, newH),
    interpolation=cv2.INTER_LINEAR
)

diff = cv2.absdiff(result_manual, result_cv2)
cv2.imwrite('bilinear_manual.jpg', result_manual)
cv2.imwrite('bilinear_cv2.jpg',    result_cv2)
cv2.imwrite('bilinear_diff.jpg',   diff)

🎨 픽셀이 특정 범위에만 몰려 있을 때 → 0~255 전체로 고르게 펼쳐주는 것!

과제 · 히스토그램 평활화 직접 구현
제출: 파이썬 1개 + 히스토그램 비교 이미지 + 결과 이미지
공식: h(v) = round( (CDF(v) - CDF_min) / (CDF_max - CDF_min) × (L-1) )
import cv2
import numpy as np
import matplotlib.pyplot as plt

gray = cv2.imread('bean.jpg', cv2.IMREAD_GRAYSCALE)

# Step 1: 히스토그램 계산
hist, _ = np.histogram(gray.flatten(), bins=256, range=(0, 256))

# Step 2: CDF (누적합)
cdf = hist.cumsum()

# Step 3: CDF_min — 0인 구간 제외! / CDF_max
cdf_min = cdf[cdf > 0].min()
cdf_max = cdf.max()
L = 256

# Step 4: 공식 적용 → LUT 생성
lut = np.round(
    (cdf - cdf_min) / (cdf_max - cdf_min) * (L - 1)
).astype(np.uint8)

# Step 5: LUT를 이미지에 적용
result = lut[gray]
cv2.imwrite('he_result.jpg', result)

# Step 6: 히스토그램 전/후 비교 저장
fig, axes = plt.subplots(1, 2, figsize=(10, 3))
axes[0].hist(gray.flatten(),   bins=256, range=(0, 255))
axes[0].set_title('Before HE')
axes[1].hist(result.flatten(), bins=256, range=(0, 255))
axes[1].set_title('After HE')
plt.tight_layout()
plt.savefig('he_histograms.jpg')
plt.show()

Week 6 · Frequency Domain Filtering ⭐ 가장 중요!
핵심 요약
DFT, LPF/HPF, Butterworth, Gaussian, Laplacian, Homomorphic 필터링을 다룹니다.
⭐ 주파수 도메인 필터링 전체 절차
1
Zero-padding: 이미지 크기를 2M × 2N으로 확장 (경계 효과 방지)
2
Centering: (-1)^(x+y) 곱해서 저주파를 중심으로 이동
3
DFT: 공간 도메인 → 주파수 도메인 변환
4
Filtering: G(u,v) = H(u,v) × F(u,v)
5
Inverse DFT: 주파수 도메인 → 다시 공간 도메인으로
⭐ 필터 공식 정리 — 암기 필수
Ideal LPF / HPF
LPF: H(u,v) = 1 if D(u,v) ≤ D0, else 0 HPF: H(u,v) = 1 - H_lpf(u,v)
Butterworth LPF (order n)
H(u,v) = 1 / (1 + (D(u,v)/D0)^(2n)) → 절단 주파수에서 값이 0.5
Gaussian LPF
H(u,v) = e^(-D²(u,v) / 2D0²)
Laplacian (주파수 도메인)
H(u,v) = -(u² + v²) 샤프닝: f(x,y) - g(x,y) 블러링: f(x,y) + g(x,y)
🔬 Homomorphic Filtering — 어려운 내용
💡 이미지 = 조명(저주파) × 반사율(고주파). 로그를 씌워 곱셈→덧셈으로 바꿔서 각각 처리!
1
f(x,y) = i(x,y) × r(x,y) → 로그: ln(f) = ln(i) + ln(r)
2
DFT 적용 후 필터링 (조명 약화↓ + 반사율 강화↑)
3
Inverse DFT → 지수함수(exp)로 복원
H(u,v) = (γH - γL)(1 - e^(-cD²/D0²)) + γL γL < 1, γH > 1

실습 1 · DFT 전체 파이프라인 + 스펙트럼 시각화
Zero-padding → Centering → DFT → 로그 스케일 시각화
import cv2
import numpy as np

img = cv2.imread('sample.jpg', cv2.IMREAD_GRAYSCALE).astype(float)
M, N = img.shape

# Step 1: Zero-padding (2M × 2N)
fp = np.zeros((2*M, 2*N))
fp[:M, :N] = img

# Step 2: Centering — (-1)^(x+y) 곱하기
xi  = np.arange(2*M).reshape(-1, 1)
yi  = np.arange(2*N).reshape(1, -1)
fpc = fp * ((-1) ** (xi + yi))

# Step 3: DFT
F = np.fft.fft2(fpc)

import cv2
import numpy as np

def ideal_lpf(shape, D0):
    M, N = shape
    u = np.arange(M) - M // 2
    v = np.arange(N) - N // 2
    V, U = np.meshgrid(v, u)
    return (np.sqrt(U**2 + V**2) <= D0).astype(float)

img = cv2.imread('sample.jpg', cv2.IMREAD_GRAYSCALE).astype(float)
M, N = img.shape

fp  = np.zeros((2*M, 2*N));  fp[:M, :N] = img
xi  = np.arange(2*M).reshape(-1, 1)
yi  = np.arange(2*N).reshape(1, -1)
F   = np.fft.fft2(fp * ((-1) ** (xi + yi)))

H = ideal_lpf((2*M, 2*N), D0=30)
G = F * H

g_pad = np.real(np.fft.ifft2(G)) * ((-1) ** (xi + yi))
g = np.clip(g_pad[:M, :N], 0, 255).astype(np.uint8)
cv2.imwrite('lpf_result.jpg', g)

import numpy as np

xi  = np.arange(2*M).reshape(-1, 1)
yi  = np.arange(2*N).reshape(1, -1)
F   = np.fft.fft2(fp * ((-1) ** (xi + yi)))

# Laplacian: H(u,v) = -(u² + v²)
# 중심 (M, N) 기준 좌표
u = np.arange(2*M) - M
v = np.arange(2*N) - N
V, U = np.meshgrid(v, u)
H_lap = -(U**2 + V**2)

g_pad = np.real(np.fft.ifft2(F * H_lap)) * ((-1) ** (xi + yi))
g     = g_pad[:M, :N]
g_norm = (g - g.min()) / (g.max() - g.min()) * 255

# 샤프닝: 원본 - 필터 결과
sharpened = np.clip(img - g_norm, 0, 255).astype(np.uint8)

과제 1 · BLPF vs GLPF 비교 (D0=30)
제출: 코드 1개(분석 주석 포함) + 결과 이미지 2개 (blpf_result.jpg / glpf_result.jpg)
import cv2
import numpy as np

# ── DFT 헬퍼 함수 ──────────────────────────────────────
def dft_forward(img):
    M, N = img.shape
    fp   = np.zeros((2*M, 2*N))
    fp[:M, :N] = img
    xi = np.arange(2*M).reshape(-1, 1)
    yi = np.arange(2*N).reshape(1, -1)
    F  = np.fft.fft2(fp * ((-1) ** (xi + yi)))
    return F, xi, yi, M, N

def dft_inverse(G, xi, yi, M, N):
    g_pad = np.real(np.fft.ifft2(G)) * ((-1) ** (xi + yi))
    return np.clip(g_pad[:M, :N], 0, 255).astype(np.uint8)

# ── D(u,v): 중심(M, N) 기준 거리 행렬 ────────────────
def compute_D(M, N):
    u = np.arange(2*M) - M
    v = np.arange(2*N) - N
    V, U = np.meshgrid(v, u)
    return np.sqrt(U**2 + V**2)

# ── BLPF: H = 1 / (1 + (D/D0)^(2n)) ─────────────────
def blpf(M, N, D0, n=2):
    D = compute_D(M, N)
    return 1 / (1 + (D / D0) ** (2 * n))

# ── GLPF: H = exp(-D² / 2D0²) ────────────────────────
def glpf(M, N, D0):
    D = compute_D(M, N)
    return np.exp(-D**2 / (2 * D0**2))

# ── 메인 ─────────────────────────────────────────────
img = cv2.imread('sample.jpg', cv2.IMREAD_GRAYSCALE).astype(float)
F, xi, yi, M, N = dft_forward(img)
D0 = 30

for name, H in [('blpf', blpf(M, N, D0, n=2)),
                ('glpf', glpf(M, N, D0))]:
    out = dft_inverse(F * H, xi, yi, M, N)
    cv2.imwrite(f'{name}_result.jpg', out)

# [분석 주석 — 반드시 포함]
# BLPF (n=2, D0=30):
#   엣지 경계가 비교적 선명하게 유지됨.
#   링잉(ringing) 아티팩트가 엣지 근처에서 약간 발생.
# GLPF (D0=30):
#   엣지가 더 부드럽게 번지고 전환이 점진적.
#   링잉 아티팩트 없음. 더 자연스러운 블러 효과.
# 결론: BLPF가 더 선명하지만 링잉 있고,
#        GLPF는 부드럽지만 흐림이 약간 더 많음.
