Week 3 · Image Processing (서브샘플링 & 양자화)
핵심 요약
이미지 해상도 줄이기(서브샘플링)와 색 단계 줄이기(양자화)를 다룹니다.
📷 서브샘플링 (Subsampling)
📷 사진을 1/2, 1/4로 작게 만드는 것. cv2.resize() 사용!
img = cv2.imread('dogs.jpg') h, w = img.shape[:2] # 1/2로 줄이기 half = cv2.resize(img, (w//2, h//2)) # 1/4로 줄이기 quarter = cv2.resize(img, (w//4, h//4)) cv2.imwrite('half.jpg', half)
과제: dogs.jpg를 2, 4, 8배로 서브샘플링 → 파이썬 1개 + 이미지 3개 제출
🎨 양자화 (Quantization)
🎨 256색(0~255)을 4색, 8색처럼 줄이는 것. 계단 현상 발생!
img = cv2.imread('xray.jpg', cv2.IMREAD_GRAYSCALE) levels = 4 # 4단계로 줄이기 # Step 1: 그룹 크기 계산 group_size = 256 // levels # = 64 # Step 2: 그룹화 + 0~255로 스케일링 quantized = (img // group_size) * (255 // (levels-1)) # Step 3: uint8 변환 quantized = quantized.astype(np.uint8)
과제: xray.jpg를 32, 8, 4, 2단계로 양자화 → 이미지 4개 제출

Week 4 · Sampling & Upsampling (보간법)
핵심 요약
Nyquist 정리, Zone Plate, 보간법(Nearest Neighbor / Bilinear / Bicubic)을 다룹니다.
🌊 저주파 vs 고주파
🌊 저주파 = 천천히 변하는 부분 (배경, 하늘) / 고주파 = 빠르게 변하는 부분 (엣지, 경계선)
⭐ Nyquist 정리
Stride(간격)가 Nyquist frequency보다 크면 → 앨리어싱(aliasing) 발생! 정보 왜곡!
Zone Plate 이미지로 확인: stride를 크게 하면 바깥쪽(고주파) 영역이 뭉개짐
⭐ 보간법 (Interpolation) 핵심
📸 작은 이미지를 크게 키울 때 없는 픽셀을 채우는 방법!
Nearest Neighbor
가장 가까운 픽셀 값 그대로 복사. 빠르지만 계단 현상
Bilinear
주변 4개 점을 선형 보간. 더 부드러운 결과
Bicubic
주변 16개 점으로 3차 함수 보간. 가장 부드러움
OpenCV 코드
cv2.resize(img, (w,h),
interpolation=cv2.INTER_LINEAR)

Week 5 · Histogram & Contrast Enhancement
핵심 요약
히스토그램, 감마 보정, 대비 향상, 히스토그램 평활화를 다룹니다.
📊 히스토그램
📊 히스토그램 = 픽셀 값(0~255)별로 몇 개의 픽셀이 있는지 세어 그래프로 나타낸 것
import matplotlib.pyplot as plt gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY) plt.hist(gray.flatten(), bins=256, range=(0,255)) plt.show()
💡 감마 보정 (Gamma Correction)
💡 픽셀 값을 지수함수로 변환해 밝기 조절. 히스토그램이 한쪽에 몰린 경우에만 효과적!
gamma = 0.5 # < 1: 밝게, > 1: 어둡게 corrected = np.power(img/255.0, gamma) * 255 corrected = corrected.astype(np.uint8)
⭐ 히스토그램 평활화 (Histogram Equalization)
🎨 픽셀이 특정 범위에만 몰려 있을 때 → 0~255 전체로 고르게 펼쳐주는 것!
h(v) = round( (cdf(v) - cdf_min) / (cdf_max - cdf_min) × (L-1) )
# 수동 구현 (과제 - 내장함수 사용 금지!) hist, _ = np.histogram(gray.flatten(), 256, [0,256]) cdf = hist.cumsum() cdf_min = cdf[cdf > 0].min() cdf_max = cdf.max() L = 256 lut = np.round( (cdf - cdf_min) / (cdf_max - cdf_min) * (L-1) ).astype(np.uint8) result = lut[gray] # LUT 적용
⚠️ 경고: cv2.equalizeHist() 사용하면 미제출 처리!

Week 6 · Frequency Domain Filtering ⭐ 가장 중요!
핵심 요약
DFT, LPF/HPF, Butterworth, Gaussian, Laplacian, Homomorphic 필터링을 다룹니다.
⭐ 주파수 도메인 필터링 전체 절차
1
Zero-padding: 이미지 크기를 2M × 2N으로 확장 (경계 효과 방지)
2
Centering: (-1)^(x+y) 곱해서 저주파를 중심으로 이동
3
DFT: 공간 도메인 → 주파수 도메인 변환
4
Filtering: G(u,v) = H(u,v) × F(u,v)
5
Inverse DFT: 주파수 도메인 → 다시 공간 도메인으로
⭐ 필터 공식 정리 — 암기 필수
Ideal LPF / HPF
LPF: H(u,v) = 1 if D(u,v) ≤ D0, else 0 HPF: H(u,v) = 1 - H_lpf(u,v)
Butterworth LPF (order n)
H(u,v) = 1 / (1 + (D(u,v)/D0)^(2n)) → 절단 주파수에서 값이 0.5
Gaussian LPF
H(u,v) = e^(-D²(u,v) / 2D0²)
Laplacian (주파수 도메인)
H(u,v) = -(u² + v²) 샤프닝: f(x,y) - g(x,y) 블러링: f(x,y) + g(x,y)
🔬 Homomorphic Filtering — 어려운 내용
💡 쉽게 말하면: 이미지 = 조명(저주파) × 반사율(고주파). 로그를 씌워 곱셈→덧셈으로 바꿔서 각각 처리!
1
f(x,y) = i(x,y) × r(x,y) → 로그: ln(f) = ln(i) + ln(r)
2
DFT 적용 후 필터링 (조명 약화↓ + 반사율 강화↑)
3
Inverse DFT → 지수함수(exp)로 복원
H(u,v) = (γH - γL)(1 - e^(-cD²/D0²)) + γL γL < 1, γH > 1
